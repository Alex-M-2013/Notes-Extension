# Notes Extension

A simple note-taking extension for Firefox. <br>

__To install:__
- Go to the releases tab and download the [latest version](https://github.com/Alex-M-2013/Notes-Extension/releases/latest) (the .xpi file)
- In Firefox, go to [about:addons](about:addons)
- Click the gear icon and click 'Install Add-on From File'
- Select the .xpi file
